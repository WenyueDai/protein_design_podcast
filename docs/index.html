Protein Design Podcast

A daily automated digest of new papers on protein design, antibody engineering, and enzyme design. A pipeline runs every morning, ranks new papers from 42 sources, and narrates them into a ~60-minute episode.

ⓘ Built on free resources only — audio quality is limited. Use it to spot papers worth reading, not as a substitute for reading them.
